Guide

Building Websites on Android: Termux + Antigravity CLI Guide

Developing modern web applications no longer requires a bulky laptop. By combining Termux (a powerful Linux terminal emulator for Android) and Google Antigravity (AGY), you can turn your phone into a complete developer workstation with AI pair-programming support.

01

Setting Up Termux on Android

Avoid the Google Play Store version as it is deprecated. Instead, download the latest version from F-Droid or GitHub releases.

Once installed, open Termux and run the following command to update core directories and install basic utilities:

pkg update && pkg upgrade
pkg install git nodejs-lts python build-essential openssh

💡 Tip: Run termux-setup-storage to grant file permission so you can access your phone’s internal storage.

02

Setting Up Your Workspace

Clone your Angular codebase, navigate inside it, and install all dependencies:

# Clone code
git clone git@github.com:your-username/website.git
cd website

# Install dependencies
npm install

To run the development server locally on your phone, execute:

npm run start

Open browser at http://localhost:4200 on your Android device to preview the live site.

03

Deploying with Google Antigravity

Antigravity acts as an advanced pair-programming agent inside your Termux environment, helping you write code, troubleshoot vulnerabilities, and run builds seamlessly.

To deploy changes to GitHub Pages, run your local deploy script:

./deploy.sh

This script automatically builds the production bundle and moves it to the deployment directory (e.g., docs/). Then commit and push:

git add .
git commit -m "feat: design upgrade via Antigravity"
git push origin main
04

Unit Testing Headless on Termux

Since Termux is a CLI environment without a desktop browser, you must run your unit specs in headless mode (e.g., ChromeHeadless). Our CI pipeline automates this step on every GitHub push, ensuring your tests pass before going live.

# Headless unit testing command
npm test -- --watch=false --browsers=ChromeHeadless
பயிற்சி கையேடு

ஆண்ட்ராய்டில் வெப்சைட் உருவாக்குவது எப்படி: Termux + Antigravity கையேடு

இப்போது வெப்சைட் அல்லது செயலிகளை உருவாக்க உங்களிடம் கணினி இருக்க வேண்டிய அவசியமில்லை. Termux (ஆண்ட்ராய்டுக்கான லினக்ஸ் டெர்மினல்) மற்றும் Google Antigravity (AGY) ஆகியவற்றை இணைத்து, உங்கள் மொபைல் போனை ஒரு முழுமையான டெவலப்பர் கம்ப்யூட்டராக மாற்றலாம்.

01

ஆண்ட்ராய்டில் Termux அமைத்தல்

Play Store-இல் உள்ள பதிப்பு தற்போது காலாவதியானது. எனவே, எஃப்-டிராய்டு (F-Droid) அல்லது GitHub-லிருந்து Termux-ஐப் பதிவிறக்கம் செய்யவும்.

நிறுவிய பின், Termux-ஐத் திறந்து, தேவையான பேக்கேஜ்களை நிறுவ பின்வரும் கட்டளையை இயக்கவும்:

pkg update && pkg upgrade
pkg install git nodejs-lts python build-essential openssh

💡 குறிப்பு: உங்கள் மொபைல் மெமரியை அணுக termux-setup-storage கட்டளையை இயக்கவும்.

02

கோடிங் வொர்க்ஸ்பேஸ் அமைத்தல்

உங்கள் Angular குறியீட்டைப் பதிவிறக்கி, அதற்குள் சென்று, தேவையான ஃபைல்களை நிறுவவும்:

# குறியீட்டைப் பெறுக
git clone git@github.com:your-username/website.git
cd website

# டிபென்டென்சிகளை நிறுவ
npm install

உள்ளூர் வெப் சர்வரைத் தொடங்க பின்வருமாறு இயக்கவும்:

npm run start

உங்கள் மொபைல் பிரவுசரில் http://localhost:4200 பக்கத்தைத் திறந்து வெப்சைட்டைப் பார்வையிடலாம்.

03

Google Antigravity கொண்டு பதிவேற்றுதல்

Antigravity என்பது உங்கள் Termux சூழலில் இருந்து கொண்டு உங்களுக்குக் கோடிங் செய்ய, பிழைகளைத் திருத்த, மற்றும் பில்ட் எடுக்க உதவும் ஒரு மேம்பட்ட செயற்கை நுண்ணறிவு (AI) உதவியாளர் ஆகும்.

வெப்சைட்டை லைவாக GitHub பக்கங்களில் வெளியிட, பில்ட் ஸ்கிரிப்ட்டை இயக்கவும்:

./deploy.sh

இது உங்கள் குறியீட்டைத் தானாகவே பில்ட் செய்து docs/ கோப்புக்குள் நகர்த்தும். பின் கமிட் செய்து புஷ் செய்யவும்:

git add .
git commit -m "feat: design upgrade via Antigravity"
git push origin main
04

டெர்மினலில் யூனிட் டெஸ்டிங்

பிரவுசர் இல்லாத டெர்மினல் என்பதால், நீங்கள் ஹெட்லெஸ் முறையில் டெஸ்ட்களை ரன் செய்ய வேண்டும். கமிட் செய்யும் போது எங்களது GitHub Actions தானாகவே டெஸ்ட்களை சரிபார்த்துக்கொள்ளும்.

# டெஸ்ட் கட்டளை
npm test -- --watch=false --browsers=ChromeHeadless
